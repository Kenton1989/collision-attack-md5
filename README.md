# collision-attack-md5
This repos is used to explore the collision attack of MD5

## Compile and Run
If gcc is available:
``` bash
gcc -o main *.c
./main
```
The commands above can be combined in one line:

### CMake >= 3.12
Under project directory:
``` 
cmake .
make
```
